# 영어회화 MP3 플레이어 웹앱

이 폴더는 영어회화 연습용 MP3 파일을 로컬에서 재생하는 간단한 웹앱을 만들어 보기 위한 공간입니다.

카페에서는 소리 내어 말하기나 STT 연습이 어렵기 때문에, 집에서 회화 음원을 듣고 따라 말하는 연습을 보조하는 것이 목표입니다.

## 목표

- 출판사에서 무료로 제공하는 MP3 파일을 로컬 폴더에 저장해서 재생하기
- 별도의 음원 관리 화면 없이, 정해진 폴더의 파일을 재생 대상으로 사용하기
- 저작권 문제를 피하기 위해 음원 파일은 직접 포함하지 않고 개인 로컬 파일만 사용하기
- 밝고 단순한 화면으로 영어회화 연습에 집중할 수 있게 만들기
- HTML, CSS, JavaScript만 사용해서 가볍게 시작하기

## 폴더 구성

```text
english-conversation-mp3-player/
├── index.html
├── style.css
├── script.js
├── README.md
└── audio/
    ├── 001.mp3
    ├── 002.mp3
    └── ...
```

## MP3 파일 저장 위치

출판사에서 무료로 제공하는 132개의 MP3 파일은 아래 폴더에 직접 저장합니다.

```text
english-conversation-mp3-player/audio/
```

예시:

```text
audio/
├── 001.mp3
├── 002.mp3
├── 003.mp3
└── ...
```

MP3 파일은 저작권 보호를 위해 이 프로젝트에 포함하지 않습니다.

## 첫 번째 버전 기능

- MP3 파일 목록 표시
- 선택한 파일 재생
- 재생 / 일시정지 버튼
- 이전 파일 / 다음 파일 버튼
- 현재 재생 중인 파일 이름 표시
- 재생 위치 이동
- 재생 속도 선택
- 한 파일 반복
- 파일 이름 검색
- 브라우저에서 폴더 목록을 읽지 못할 때 사용할 파일 선택 버튼
- 밝고 심플한 화면 디자인

## 실행 방법

이 폴더에서 로컬 서버를 실행합니다.

```bash
python3 -m http.server 8100
```

브라우저에서는 아래 주소로 접속합니다.

```text
http://localhost:8100
```

브라우저에서 접속하면 `audio/` 폴더의 MP3 파일을 자동으로 읽어 목록에 표시합니다.
